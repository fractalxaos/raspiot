Push Button Monitor

Demonstrates web GPIO control by monitoring the number of times a push button, connected to the Raspberry Pi GPIO, has been pressed. Also starts the push button agent background process, if it is not running. The agent launches a callback function that updates the pushbutton count, stored in a file, each time the button gets pushed.

This web page uses Javascript to periodically send XML requests for the button push count, which is stored in a dynamic JSON file on the raspbery pi.
